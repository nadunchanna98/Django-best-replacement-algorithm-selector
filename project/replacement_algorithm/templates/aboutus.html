Operating Systems (EC-6110) Group Assignment

Problem Statement:

For a given reference string (should be given as input), select the best replacement algorithm from the following set of algorithms:

  • First In First Out (FIFO)
  • Least Recently Used (LRU)
  • Least Frequently Used (LFU)
  • Most Frequently Used (MFU)
